#Lab 05 - Estructura de datos

1. Descripción del contenido del repositorio.
El código implementa operaciones básicas sobre el arreglo dinámico: crear, insertar, eliminar, obtener e imprimir.
    lab5/
        include/
            dynarray.h:Header con la definición de la estructura y funciones
        src/
            dynarray.c: Implementación del arreglo dinámico
            main.c: Programa principal de prueba
        Makefile: Archivo para compilar automáticamente
        README.md: Descripción del proyecto

2. Pasos para compilar.
Desde la carpeta principal lab5/, ejecutar:
    make
Esto genera el ejecutable:
    prueba_dyn

3. Pasos para ejecutar.
Después de compiladar, ejecutar:
    ./prueba_dyn
El programa demostrará el funcionamiento del arreglo dinámico mediante inserciones, eliminaciones e impresión del contenido.
